# Assets & Images Folder

Place your project screenshots and profile photo here.

## Recommended Files

```
images/
├── profile-photo.jpg       # Your professional photo (500x600)
├── radio-sangam-1.png      # Project screenshot 1
├── radio-sangam-2.png      # Project screenshot 2  
├── radio-sangam-3.png      # Project screenshot 3
└── preview.png             # Portfolio preview for README
```

## Image Specifications

| Image Type | Recommended Size | Format | Max File Size |
|------------|------------------|--------|---------------|
| Profile Photo | 500x600px | JPG/WebP | 200KB |
| Project Screenshots | 1200x800px | PNG/WebP | 500KB |
| Preview Image | 1200x630px | PNG | 500KB |

## Optimization Tips

1. Use [TinyPNG](https://tinypng.com) or [Squoosh](https://squoosh.app) to compress images
2. Use WebP format for better compression
3. Ensure images are not larger than necessary
4. Consider using lazy loading for better performance
